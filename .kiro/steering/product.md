# Product Overview

This is a Next.js + Tailwind CSS + TypeScript starter template designed for rapid web application development. It serves as a boilerplate for building modern React applications with pre-configured tooling and best practices.

## Purpose

Provide a production-ready foundation for Next.js projects with:

- Type-safe development environment
- Utility-first styling with Tailwind CSS
- Pre-built reusable components
- Automated code quality checks
- Testing infrastructure

## Key Features

- Pre-built component library (buttons, links, images)
- Absolute imports using `@/` prefix
- SEO optimization with sitemap generation
- SVG-as-component support
- Open Graph image generation
- Conventional commit enforcement
- Automated linting and formatting on commit
